# scRNA-seq analysis notebook

This repository contains the notebook used to generate all scRNA-seq–related figures for the manuscript (Chenglei Tian et al., 2026).

The notebook includes the following sections:
- Data preprocessing of scRNA-seq datasets from iPSC-derived islet grafts
- Cell type annotation across three biological replicates
- Generation of figures for the manuscript

The analysis focuses on validating cell type composition, with particular attention to beta cell identity and the assessment of potential contaminating cell populations.
